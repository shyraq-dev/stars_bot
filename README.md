# ⭐ Telegram Stars Сату Боты

**aiogram 3.28.2** негізіндегі Telegram Stars сату боты.
Теңге (Kaspi P2P) және TON (Telegram @Wallet) төлемдерін қабылдайды.
Екі төлем түрі де әкімші арқылы қолмен расталады.

## ⚙️ .env конфигурациясы

| Параметр | Сипаттама |
|---|---|
| `BOT_TOKEN` | @BotFather арқылы алынған токен |
| `ADMIN_IDS` | Әкімші(лер)дің Telegram ID-і, үтірмен |
| `DB_PATH` | SQLite файл жолы (әдепкі: `stars_bot.db`) |
| `CHANNEL_URL` | Жобаның ресми Telegram арнасы |
| `SUPPORT_USERNAME` | Қолдау аккаунты (@username) |

Kaspi реквизиттері **әкімші тақтасынан** орнатылады:
`/admin → 💳 Kaspi реквизиттер`

## 💳 Төлем жүйелері

### Теңге — Kaspi P2P
```
Пайдаланушы → Kaspi-ге аударады
           → PDF түбіртекті жібереді
           → Бот PDF + [✅ Растау] [❌ Қабылдамау] батырмаларымен
             әкімшіге жібереді
           → Әкімші Kaspi-ден ақша түскенін тексереді
           → ✅ Растаса — жұлдыздар бірден аударылады
           → ❌ Бас тартса — пайдаланушыға хабарлама жіберіледі
```

### TON — Telegram @Wallet P2P
```
Пайдаланушы → @Wallet арқылы TON аударады
           → Скриншот немесе txid жібереді
           → Бот дәлел + [✅ Растау] [❌ Қабылдамау] батырмаларымен
             әкімшіге жібереді
           → Әкімші @Wallet-тен аударымды тексереді
           → ✅ Растаса — жұлдыздар бірден аударылады
```

## 🔧 Әкімші тақтасы

`/admin` командасымен ашылады:

| Бөлім | Мүмкіндік |
|---|---|
| 📊 Статистика | Пайдаланушылар, табыс, жіберілген жұлдыздар |
| 💱 Бағамдар | 1 ⭐ және 1 TON-ның теңгедегі бағасы |
| 🎁 Реферал бонусы | Әр қосылған адамға берілетін бонус (₸) |
| 💳 Kaspi реквизиттер | Телефон, карта, иесі |
| 📨 Жаппай хабарлама | Барлық пайдаланушыларға хабар жіберу |
| 📋 KZT/TON тапсырыстар | Күтудегі тапсырыстарды растау / қабылдамау |
| 👥 Қолданушыларды басқару | Бұғаттау, бұғатты алу, бұғатталғандар тізімі |

---

## 🎁 Реферал жүйесі

- Әр пайдаланушының бірегей реферал сілтемесі бар
- Дос сілтеме арқылы тіркеліп, **алғаш рет сатып алса**
- Шақырушыға **бонус (₸)** беріледі
- Бонус мөлшері: `/admin → 🎁 Реферал бонусы`

---

## 💰 Баға есебі

```
Теңге сомасы = жұлдыз саны × (1 ⭐ бағамы)
TON сомасы   = Теңге сомасы ÷ (1 TON бағамы)
```

Бағамдарды `/admin → 💱 Бағамдар` арқылы өзгертуге болады.
